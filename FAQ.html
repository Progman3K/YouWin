FAQ:

Q - What is YouWin?

A – YouWin allows you to write applications that will run on Android, iOS, OSX, linux and Windows.

Q - Is this based on the Wine project?

A – No. Wine lets you run existing x86/x86_64 Windows application binaries on Linux. YouWin lets you build native applications from source for multiple platforms.

Q – Does YouWin require me to have Windows

A – No.

Q – Why is it called YouWin

A – Developers, developers, developers; this time, You Win.

Q – Can YouWin be used to create text-mode applications?

A – Yes, with very little effort, you can write event-driven text-mode applications. (see sample)

Q – How big is YouWin?

A – Around 700 K bytes

Q – Can YouWin be embedded?

A – Yes, embedded linux and other *nix-type platforms can host YouWin apps.

Q – Memory spaces, multiple threads, multiple apps, etcetera what about them?

A – YouWin executes each application in separate memory, applications cannot read each others memory. Although you can use threads with YouWin, the handles returned should not be shared between threads.
